# Mint dApp — versión ethers v6

Integración cruda con `window.ethereum` + ethers v6. Ver el [README del proyecto](../README.md) para la explicación de los conceptos.

Necesitás MetaMask. La lógica está en [`app/page.jsx`](app/page.jsx); el address y el ABI en [`lib/contract.js`](lib/contract.js) (reemplazá el `CONTRACT_ADDRESS` por uno real para mintear de verdad).
